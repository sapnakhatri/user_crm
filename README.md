## About Project

- Usercrm is an application which has create user REST API
- This user API maintain history (audit log) asynchronously so that as soon as
user is created then response will be send.
- Laravel Queue is used along with worker to maintain the history.
- If there is any exception in worker then this worker will retry to maintain history.
- User API is accepting inputs with proper validations.
- API is returning meaningful validation message with proper status code.
- API also have rate limiting in place.
